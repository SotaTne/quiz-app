---
title: 総和プログラムの停止性
---

| id | question | answer | explanation |
|----|----------|--------|-------------|
| termination-001 | 停止性の証明で使う、自然数のような順序付けられた集合を何というか。 | 整礎集合 | 無限に減少し続ける列がない。 |
| termination-002 | ループが停止することを示すため、各時点に割り当てる関数を何というか。 | ランキング関数 | ranking function。 |
| termination-003 | 総和プログラムのランキング関数 r(i) は何か。 | r(i)=n-i+1 | iはループのインデックス。 |
| termination-004 | ループの次の回でランキング関数はどう変化するか。 | 1減少する | r(k+1)=r(k)-1。 |
| termination-005 | 自然数上で値が減少し続けられない性質が、何の証明につながるか。 | 停止性 | ループが無限に続かないことを示す。 |
| termination-006 | r(k)=n-k+1 の k は何を表すか。 | ループの回数・インデックス | 現在の反復位置。 |
| termination-007 | ランキング関数の値が属する集合は何か。 | 自然数 | 整礎集合の例。 |
| termination-008 | r(k+1)=r(k)-1 は、値がどうなることを表すか。 | 1減少する | 次の反復で小さくなる。 |
| termination-009 | 自然数上で値が減少し続けられない性質が保証するものは何か。 | 停止性 | 無限降下がない。 |
| termination-010 | 停止性の証明で比較する現在と次の値は何か。 | r(k)とr(k+1) | 次の値が小さいことを示す。 |
| termination-011 | 無限に減少し続ける数列を何というか。 | 無限降下列 | 整礎集合には存在しない。 |
| termination-012 | ループの各反復で追跡する、停止性の証明用の量は何か。 | ランキング関数 | 値の減少を確認する。 |
