---
title: ソフトウェア工学 オンライン OSSと商用ソフトウェア
---

| id | question | answer | explanation |
|----|----------|--------|-------------|
| seonline-001 | OSSは何の略か。 | Open Source Software | オープンソースソフトウェア。 |
| seonline-002 | OSSの定義を策定する代表的な団体は何か。 | OSI | Open Source Initiative。 |
| seonline-003 | OSSでは利用者が入手できることが原則となるものは何か。 | ソースコード | 調査・変更の基礎になる。 |
| seonline-004 | OSSでソースコードを変更して作ったソフトウェアを何というか。 | 派生物 | derivative work。 |
| seonline-005 | OSSの定義に含まれる、利用者やグループへの差別をしてはいけない原則は何か。 | 差別の禁止 | 個人・グループ・利用分野を制限しない。 |
| seonline-006 | 無料ではなく「自由」を意味するFree Softwareを提唱した人物は誰か。 | Richard Stallman | FSFの創設者。 |
| seonline-007 | Free Softwareの自由には、コピー・研究・変更と何が含まれるか。 | 配布 | 自由を制限なく利用できることが重要。 |
| seonline-008 | 著作物を利用する条件を定めるものは何か。 | ライセンス | 利用・改変・再配布の条件を定める。 |
| seonline-009 | 著作権を誰も持たず自由に利用できる状態を何というか。 | パブリックドメイン | 著作権保護期間満了など。 |
| seonline-010 | OSSと商用ソフトウェアを区別する観点を1つ答えよ。 | ソースコードの公開 | 開発者、品質、著作権、ライセンスなども観点。 |
| seonline-011 | OSSでソースコードを入手できることを何というか。 | ソースコードへのアクセス | 改変・研究の前提。 |
| seonline-012 | OSSで変更物を再配布できる自由を何というか。 | 再頒布の自由 | license distribution。 |
| seonline-013 | OSSで個人やグループを差別してはいけない原則は何か。 | 差別の禁止 | 誰でも利用できる。 |
| seonline-014 | GPLなどの自由ソフトウェアを推進した団体は何か。 | Free Software Foundation | FSF。 |
| seonline-015 | 著作物の利用条件を定めるものは何か。 | ライセンス | 利用・改変・配布を定める。 |
| seonline-016 | 著作権者に無断で一定範囲の利用が認められる例は何か。 | 私的利用 | 法律上の例外。 |
| seonline-017 | 他人の著作物を紹介するために一部利用することは何か。 | 引用 | 自分の著作物が主である必要がある。 |
| seonline-018 | 著作権を誰も持たず自由に利用できる状態は何か。 | パブリックドメイン | 保護期間満了など。 |
